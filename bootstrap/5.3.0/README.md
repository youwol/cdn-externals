# Wrapper of the npm package [bootstrap](https://www.npmjs.com/package/bootstrap) for publication in YouWol's CDN.




This library is part of the hybrid cloud/local ecosystem 
[YouWol](https://platform.youwol.com/applications/@youwol/platform/latest).

## Links

[Developers documentation](https://platform.youwol.com/applications/@youwol/cdn-explorer/latest?package=bootstrap)


[Sources](https://github.com/youwol/cdn-externals/tree/master/bootstrap)

# Installation, Build, Test

To install the required dependencies:

```shell
yarn
```
---
To build for development:

```shell
yarn build:dev
```

To build for production:

```shell
yarn build:prod
```
