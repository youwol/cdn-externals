# Wrapper of the npm package [@popperjs/core](https://www.npmjs.com/package/@popperjs/core) for publication in YouWol's CDN.




This library is part of the hybrid cloud/local ecosystem 
[YouWol](https://platform.youwol.com/applications/@youwol/platform/latest).

## Links

[Developers documentation](https://platform.youwol.com/applications/@youwol/cdn-explorer/latest?package=rxjs)


[Sources](https://github.com/youwol/cdn-externals/tree/master/rxjs)

# Installation, Build, Test

To install the required dependencies:

```shell
yarn
```
---
To build for development:

```shell
yarn build:dev
```

To build for production:

```shell
yarn build:prod
```
